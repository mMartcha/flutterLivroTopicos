# Trabalho Prático — Full Stack com Dart

Projeto desenvolvido para a disciplina **Tópicos Especiais** por Marcello Stefenon Filho.

## Descrição

Este projeto consiste em uma aplicação full stack com backend em Dart, documentação no Postman e frontend em Flutter.

O tema escolhido foi **Biblioteca**, com duas entidades em relacionamento **1:N**:

- **Autor**: entidade pai
- **Livro**: entidade filho

Um autor pode possuir vários livros, e cada livro pertence a um único autor. Isso atende ao requisito do enunciado de trabalhar com exatamente duas entidades e relacionamento 1:N. :contentReference[oaicite:1]{index=1}
